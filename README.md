# analysis-everpeak
# Análisis de PM2.5 en Latinoamérica - EverPeak

## Descripción del Proyecto
Este proyecto analiza los niveles de PM2.5 (partículas finas) en países de Latinoamérica utilizando Python y técnicas de análisis de datos.

## Objetivos
- Analizar la calidad del aire en países latinoamericanos
- Identificar patrones y tendencias en los niveles de contaminación
- Proporcionar insights para políticas ambientales

## Tecnologías Utilizadas
- Python 3.x
- Pandas para manipulación de datos
- Matplotlib y Seaborn para visualización
- Jupyter Notebook para análisis interactivo
